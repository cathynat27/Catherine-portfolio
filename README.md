# Catherine Natukunda - Portfolio Website

A modern, responsive personal portfolio website built with React.js and Tailwind CSS, featuring smooth animations and dark/light mode toggle.

## Features

- **Responsive Design**: Fully responsive across all device sizes
- **Dark/Light Mode**: Toggle between themes with smooth transitions
- **Smooth Animations**: Powered by Framer Motion
- **Modern UI**: Clean, professional design suitable for recruiters
- **Performance Optimized**: Fast loading and smooth interactions

## Sections

1. **Hero Section**: Introduction with call-to-action buttons
2. **About Section**: Story-driven background and journey
3. **Skills Section**: Categorized technical skills
4. **Projects Section**: Detailed project showcases
5. **Experience Section**: Professional work history
6. **Certifications Section**: Educational achievements
7. **Contact Section**: Professional contact information

## Technologies Used

- React.js
- Tailwind CSS
- Framer Motion
- Lucide React (icons)

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Building for Production

```bash
npm run build
```

This builds the app for production to the `build` folder.

## Customization

### Personal Information
Update the portfolio data in `src/data/portfolioData.js` to customize:
- Personal details
- About section content
- Skills and technologies
- Project information
- Work experience
- Certifications

### Styling
- Colors and themes can be modified in `tailwind.config.js`
- Custom styles are in `src/index.css`

### Components
All components are modular and located in `src/components/`:
- `Header.js` - Navigation and theme toggle
- `Hero.js` - Landing section
- `About.js` - About section
- `Skills.js` - Skills showcase
- `Projects.js` - Project portfolio
- `Experience.js` - Work history
- `Certifications.js` - Certifications display
- `Contact.js` - Contact information

## Deployment
